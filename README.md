# PI_Vendas_de_Carros

## 🚗 Sobre o Projeto

**Este é o Car Shop, um e-commerce automotivo desenvolvido como Projeto Integrador (PI) para o curso de Técnico de informática.O objetivo principal do sistema é conectar compradores e entusiastas de veículos a um catálogo digital moderno, responsivo e de alta performance. A plataforma simula a experiência real de uma concessionária digital ou marketplace de carros, permitindo a visualização de ofertas, detalhes técnicos dos veículos e interface de navegação fluida.**

## 💡 Principais funcionalidades

1. **Catálogo Dinâmico:** Apresentação automática de ofertas especiais na página inicial com transições rápidas.
2. **Carrossel de Destaques:** Apresentação automática de ofertas especiais na página inicial com transições rápidas.
3. **Interface Responsiva:** Navegação otimizada para computadores, tablets e celulares utilizando o framework Bootstrap.
4. **Identidade Visual Premium:** Layout moderno com tema focado no contraste esportivo (Preto, Branco e Vermelho).

## 🛠️ Tecnologias Utilizadas

1. **HTML5** - Estruturação semântica das páginas.
2. **CSS3** - Estilização personalizada de componentes.
3. **Bootstrap 5.3** - Framework para layout responsivo, sistema de grids, navbars e componentes de carrossel.
